## Description

This is a repository developed with the purpose of having a starting point in future projects.

## Installation

You will also need to install and start a MongoDB instance locally. If not, please look at the dockerized version. (soon)

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
